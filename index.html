HTML

HTML é a linguagem usada para criar páginas na web, definindo sua estrutura incluindo textos, imagens, links, etc e conteúdo através demarcações chamadas de "tags".

CSS

CSS é uma linguagem usada para estilizar páginas web, controlando aspectos visuais como cores,fontes e layout do conteúdo HTML.

JavaScript

JavaScript é uma linguagem de programação usada para criar interatividade em páginas da web, permitindo manipular conteúdo, responder a eventos do usuário e criar animações.

PHP

PHP é uma linguagem de programação para criar aplicativos web dinâmicos e interativos. Permite processar formulários, acessar bancos de dados e gerar conteúdo dinâmico.

Python

Python é uma linguagem de programação, conhecida por sua sintaxe simples e legível. É amplamente utilizada em diversas áreas, como desenvolvimento web, ciência de dados e automação. Sua grande comunidade e vasta coleção de bibliotecas a tornam popular para uma variedade de aplicações

MySql

MySQL é um sistema de gerenciamento de banco de dados relacional. Ele utiliza a linguagem SQL para manipular dados em tabelas relacionadas e é conhecido por sua confiabilidade e desempenho. É uma escolha popular para uma variedade de aplicativos, desde sites simples até sistemas corporativos complexos.
